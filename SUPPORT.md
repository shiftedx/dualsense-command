# Support

Thanks for trying DualSense Command Center.

## Bugs

Use [GitHub Issues](https://github.com/shiftedx/dualsense-command/issues) for
bugs, crashes, installer problems, controller detection problems, and game
telemetry issues.

Please include:

- DSCC version.
- Controller model.
- USB or Bluetooth.
- Windows version.
- Game name.
- Whether telemetry is connected.
- A short description of what happened.

## Ideas And Feedback

Use Issues for now. GitHub Discussions is the best long-term place for general
feedback, tuning ideas, and setup questions, but it must be enabled in the repo
settings by a maintainer first.

## Before Posting

Check [Troubleshooting](docs/troubleshooting.md) first. It covers SmartScreen,
Forza Data Out, LAN Access, controller detection, and Steam Input basics.
